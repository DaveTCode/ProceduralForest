#Requirements

- python 3
- pygame (find for windows and python 3.4 [here](http://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame))

#Run

Run the application from the command line by specifying a seed. 

```
python run.py --seed <n>
```
